Help & Shortcuts

Shortcuts:

  • Enter - Next match
  • Shift+Enter - Previous match
  • Ctrl+Shift+F - Toggle search

Notes:

  • Click the extension icon to open floating search window
  • Floating window can be dragged and stays open on the page
  • All search functions work in both popup and floating modes

Regex Examples:

  • \d+ - Match numbers
  • \b\w+@\w+\.\w+\b - Match emails
  • https?://\S+ - Match URLs
